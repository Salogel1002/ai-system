# ai-system

AI system architecture with memory, skills, and cost control.

## Purpose

This repository documents and structures a modular AI system with:

- thin skills
- controlled memory via mem0
- vector storage via Qdrant Cloud
- clear runtime patterns
- explicit cost control
- documented architecture decisions

## Structure

- `docs/architecture` — core architecture documents
- `docs/adr` — architecture decision records
- `docs/playbooks` — operational playbooks

## Current focus

Version 1 establishes the architectural foundation before runtime, memory integration, and orchestration are implemented.
