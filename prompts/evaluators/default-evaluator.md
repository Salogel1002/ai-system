# Default Evaluator

Review the output only after the main task is complete.

## Evaluation criteria

- Is the result correct?
- Is it logically structured?
- Is it proportionate to the task?
- Was unnecessary context avoided?
- Was memory only used if needed?
- Is the output clear and actionable?

## Rule

If the answer is bloated, vague, or over-engineered, it should be revised.
