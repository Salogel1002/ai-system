# Skill Registry

Die Skill Registry ist das zentrale Verzeichnis aller Skills, die im System grundsätzlich bekannt sind. Sie ist das einzige offizielle Eingangstor für interne, externe und über Clawhub installierte Skills.

## Zweck

Dieses Dokument stellt sicher, dass kein Skill „einfach existiert“, sondern jeder Skill formal registriert, klassifiziert und kontrollierbar ist.

Ohne Skill Registry entstehen:
- unklare Skill-Bestände
- doppelte oder widersprüchliche Fähigkeiten
- unkontrollierte Tool-Nutzung
- fehlende Governance für externe Skills

## Grundprinzipien

### 1. Keine Nutzung ohne Registry
Ein Skill darf nur verwendet werden, wenn er in der Registry erfasst ist.

### 2. Installation ist nicht Freischaltung
Ein installierter Skill ist noch nicht automatisch nutzbar.

### 3. Einheitliches Innenmodell
Auch externe Skills werden auf ein internes Registry-Modell abgebildet.

### 4. Herkunft ist relevant
Die Quelle eines Skills beeinflusst Vertrauen, Review-Tiefe und Nutzungsrechte.
